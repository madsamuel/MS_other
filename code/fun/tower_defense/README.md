# Mario Tower Defense

A tower defense game set in the Super Mario universe where you defend against classic Mario enemies using Mario-themed towers.

## How to Play

1. **Open the Game**: Simply open `index.html` in a web browser
2. **New Game**: Click "New Game" to start a new game
3. **Select a Tower**: Click on a tower in the right panel to select it
4. **Place Towers**: Click on the canvas to place towers (avoid the path where enemies walk)
5. **Start Waves**: Click "Start Wave" to begin spawning enemies
6. **Survive**: Defend against all enemies in the level to progress to the next level

## Game Features

### Towers
- **Pipe (50G)**: Basic tower with steady fire rate
- **Fireball (100G)**: Increased damage with splash damage
- **Bob-omb (150G)**: Explosive tower with powerful area damage
- **Mushroom (120G)**: Fast-firing support tower

### Enemies
- **Goomba**: Weak basic enemy
- **Koopa Troopa**: Armored enemy with more health
- **Piranha Plant**: Dangerous mid-tier enemy
- **Bullet Bill**: Fast but fragile
- **Bowser Jr**: Boss enemy with high health

### Economics
- Earn gold by defeating enemies
- Gold can be used to purchase towers
- Complete levels with gold remaining to build up resources
- Start each level with bonus gold

### Levels
- 10 progressively challenging levels
- Final level features Bowser Jr as a boss enemy
- Waves increase in difficulty with each level

## Controls

- **Left Click**: Place selected tower
- **Start Wave**: Begin spawning enemies
- **Pause**: Pause the game
- **Save Game**: Save your progress to browser storage
- **Load Game**: Load a previously saved game

## Game Mechanics

1. **Health System**: You start with 20 lives. Each enemy that reaches the end costs 1 life
2. **Scoring**: Earn points by defeating enemies
3. **Tower Placement**: Towers cannot be placed on the enemy path
4. **Combat**: Towers automatically target and shoot enemies in range
5. **Persistence**: Save and load your game at any time

## Winning

Complete all 10 levels without running out of lives to win the game.

## Losing

The game ends when your lives reach 0.

## Tips

- Plan your tower placement carefully for maximum coverage
- Upgrade your tower arsenal as you progress
- Manage your gold wisely - don't spend it all at once
- Higher tier towers are more expensive but deal more damage
- Use the pause feature to plan your strategy

Enjoy defending the Mushroom Kingdom!
