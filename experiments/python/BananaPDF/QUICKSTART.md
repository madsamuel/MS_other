# BananaPDF Quick Start Guide

## Getting Started in 5 Minutes

### 1. Install Dependencies
```bash
cd BananaPDF
pip install -r requirements.txt
```

### 2. Start the Application
```bash
python app.py
```

You should see:
```
 * Running on http://127.0.0.1:5000
```

### 3. Open in Browser
Go to: `http://localhost:5000`

### 4. Upload a PDF
- Click the "Open" button, or
- Drag and drop a PDF into the viewer area

### 5. Start Editing
- **Add Text**: Click the Text button, click on the PDF, and type
- **Highlight**: Click the Highlight button and click on the PDF
- **Add Comment**: Click the Comment button and add a note
- **Rotate Page**: Click the Rotate button to turn the page
- **Delete Page**: Click Delete to remove a page

### 6. Export Your Work
Click the "Export" button to download the edited PDF

## Common Tasks

### View Different Pages
- Click thumbnails on the left, or
- Use the page selector dropdown on the right

### Zoom In/Out
- Use the zoom buttons in the toolbar, or
- Select a zoom level from the dropdown

### Rotate a Page
Select the page you want to rotate and click the Rotate button (each click rotates 90°)

### Remove a Page
Click Delete Page (you'll be asked to confirm)

### Download Your Work
Click Export to download the PDF with all your edits

## Tips & Tricks

1. **Work Incrementally**: Export often to save your work as you go
2. **Use Zooming**: Zoom in when adding precise text boxes
3. **Organize First**: Rotate and delete pages before adding annotations
4. **Check Before Export**: Review your changes before exporting

## Troubleshooting

**PDF won't upload?**
- Make sure it's a valid PDF file
- Check that the file isn't too large (50MB max)
- Try with a different PDF file

**Annotations not showing?**
- Make sure you selected a tool (they should show highlighted)
- Click on the page after selecting a tool
- Check the right panel to see your annotations

**PDF won't export?**
- Check that you have disk space
- Try a smaller PDF first
- Restart the app and try again
