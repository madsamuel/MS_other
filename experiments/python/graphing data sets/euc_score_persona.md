# EUC Score Profiles

| Persona | Summary |
|--------|---------|
| **Kiosk User** (aka *Deskless Worker*)<br>**[5–50% of IT workforce]** | This persona type needs to share a desktop that is placed in a public place. Examples include students using a shared computer in a classroom, nurses at nursing stations, and computers used for job placement and recruiting. These desktops require automatic login. Authentication can be done through certain applications if necessary. |
| **Task Worker** (aka *Data Entry Worker, Transaction Worker*)<br>**[25–80% of IT workforce]** | This persona type describes users who perform well-defined, repetitive, and delineated tasks. These tasks are performed using a limited number of applications and may involve shared devices. Users need to create simple documents, fill spreadsheets, and draft reports. Examples include administrators, receptionists, bank tellers, call center workers, and warehouse workers. |
| **Information Worker** (aka *Office Worker, Firstline Worker, Structured Task Worker, Knowledge Consumer, Integration Worker*)<br>**[25–80% of IT workforce]** | This persona type tends to work only with data and information, not ideas. They create and consume information but do not transform or manage it. They need to find facts quickly, prepare presentations, and edit, write, and process information. Examples include bank clerks, call center operators, medical staff, sales associates, factory workers, and customer service staff. Supervisory roles include shop managers, bank managers, and nursing supervisors. |
| **Knowledge Worker** (aka *Knowledge Broker, Expert, Collaboration Worker*)<br>**[10–50% of IT workforce]** | This persona type describes people focused on research, creation, or re-use of knowledge by creating complex documents, presentations, and spreadsheets. They develop and improve processes and forms, encourage collaboration, and create workspace environments. They also create, consume, transform, and analyze data. Examples include middle/senior managers, accountants, sales managers, consultants, research analysts, teachers, and financial analysts. |
| **Power User** (aka *Developer, Data Analyst, Knowledge Generator*)<br>**[5–50% of IT workforce]** | This persona type includes people whose activities are very compute intensive and involve using many applications simultaneously. Applications consume significant CPU, disk, memory, network resources, and shared GPUs. Typical work includes visualizing models created by CAD/CAM professionals or marketing materials produced by media designers, often applying small changes without full re-rendering. Examples include engineers, scientists, architects, educators, researchers, product marketing managers, financial traders, software developers, data analysts, and statisticians. |
| **CAD/CAM Professional**<br>**[5–25% of IT workforce]** | This persona type uses graphically intensive applications for computer-aided design (CAD) and computer-aided manufacturing (CAM). Scenarios include handling 3D models with millions of elements, consuming significant CPU, disk, memory, and GPU resources at a given time. |
| **Media Designer**<br>**[5–50% of IT workforce]** | This persona type creates the overall look and feel of interactive communication products. They use text, data, graphics, sound, animation, and digital visual effects to plan, visualize, and create marketing materials based on business needs. Examples include artists, graphic designers, web designers, animators, and video editors. |
| **Mobile Worker** (aka *Work-from-Home User*)<br>**[5–80% of IT workforce]** | This persona type is typically combined with one of the other personas. It focuses on users who work from multiple locations during a day and often use multiple devices. Examples include medical personnel, pilots and cabin crew, field sales professionals, railway staff, and executives. |
